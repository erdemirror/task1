Datatypes

JavaScript 8 ширхэг өгөгдлийн төрөлтэй:

  1. String - Example: "Hello"
  2. Number - Example: 42, 3.14
  3. BigInt - Example: 12345678901234567890n
  4. Boolean - Example: true, false
  5. Undefined - Example: let a;
  6. Null - Example: let x = null;
  7. Symbol - Example: Symbol("id")
  8. Object - Example: { name: "Erdem", age: 20 }

async and await

"async and await нь promise-г бичихэд илүү амархан болгодог"
async makes a function return a Promise
await makes a function wait for as Promise

async syntax:

async function myFunction() { 
    return "Hello"; 
}
function myFunction() {
    return Promise.resolve("Hello");
}

await syntax:

The await нь зөвхөн async функц дотор ашиглагдана. Функцийг түр зогсоож promise-г биелэхийг хүлээнэ. 

async function myDisplay() {
    let myPromise = new Promise(function(resolve, reject) {
        resolve("I love You !!");
    });
    document.getElementById("demo").innerHTML = await myPromise;
}

myDisplay();

Functions

JavaScript функц нь даалгаврыг гүйцэтгэхэд зориулагдсан кодын блок.

Дуудагдаж байж ажиллана.

// 2 тоо үржих
function myFunction(p1, p2) {
    return p1 * p2;
}
// F to C
function toCelsius(fahrenheit) {
    return (5/9) * (fahrenheit-32);
}
// хувьсагчид утга аа олгох 
let value = toCelsius(100);

Функцуудыг мэддэг гэж бодсон ч мэддэггүй байж ээ. Arrow Функцийг бичихийн хажуугаар бусад бичиглэлүүдийн ялгааг оруулахаар шийдлээ. 

// Constructor
const multiply = new Function("x", "y", "return x * y");

// Declaration
function multiply(x, y) {
    return x * y;
} // No need for semicolon here

// Expression; the function is anonymous but assigned to a variable
const multiply = function (x, y) {
    return x * y;
};

// Expression; the function has its own name
const multiply = function funcName(x, y) {
    return x * y;
};

// Arrow function
const multiply = (x, y) => x * y;

// Method
const obj = {
    multiply(x, y) {
        return x * y;
    },
};
