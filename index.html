Datatypes

JavaScript has 8 primary data types. These are used to store different kinds of values:

  1. String - Example: "Hello"
  2. Number - Example: 42, 3.14
  3. BigInt - Example: 12345678901234567890n
  4. Boolean - Example: true, false
  5. Undefined - Example: let a;
  6. Null - Example: let x = null;
  7. Symbol - Example: Symbol("id")
  8. Object - Example: { name: "Erdem", age: 20 }

Functions

A JavaScript function is a block of code designed to perform a task.

It is executed when "something" invokes (calls) it.

// Multiply two numbers
function myFunction(p1, p2) {
  return p1 * p2;
}
// Convert Fahrenheit to Celsius
function toCelsius(fahrenheit) {
  return (5/9) * (fahrenheit-32);
}

let value = toCelsius(100);

Promises

A Promise is a JavaScript object that represents the eventual completion (or failure) of an asynchronous operation and its resulting value.

Creating a Promise


const myPromise = new Promise((resolve, reject) => {
  let success = true;
  if (success) {
    resolve("Success!");
  } else {
    reject("Error occurred.");
  }
});

Using .then() and .catch()


myPromise
  .then(result => {
    console.log("Success:", result);
  })
  .catch(error => {
    console.error("Error:", error);
  });

Chaining Promises


fetch('https://api.example.com/data')
  .then(response => response.json())
  .then(data => {
    console.log("Data:", data);
    return fetch('https://api.example.com/next');
  })
  .then(next => next.json())
  .then(nextData => {
    console.log("Next data:", nextData);
  })
  .catch(error => console.error(error));

Promise.all


const p1 = fetch('/api/user');
const p2 = fetch('/api/products');

Promise.all([p1, p2])
  .then(responses => Promise.all(responses.map(r => r.json())))
  .then(([user, products]) => {
    console.log("User:", user);
    console.log("Products:", products);
  })
  .catch(error => console.error("Error in one of the requests:", error));

Why Use Promises?

  • Clean error handling with .catch()
  • Prevents "callback hell"
  • Improves readability and maintenance
