Datatypes

JavaScript 8 ширхэг өгөгдлийн төрөлтэй:

  1. String - Example: "Hello"
  2. Number - Example: 42, 3.14
  3. BigInt - Example: 12345678901234567890n
  4. Boolean - Example: true, false
  5. Undefined - Example: let a;
  6. Null - Example: let x = null;
  7. Symbol - Example: Symbol("id")
  8. Object - Example: { name: "Erdem", age: 20 }
Ерөнхий ойлгомжтой гэж бодож байна.

Functions

JavaScript функц нь даалгаврыг гүйцэтгэхэд зориулагдсан кодын блок.

Дуудагдаж байж ажиллана.

// 2 тоо үржих
function myFunction(p1, p2) {
  return p1 * p2;
}
// F to C
function toCelsius(fahrenheit) {
  return (5/9) * (fahrenheit-32);
}
// хувьсагчид утга аа олгох 
let value = toCelsius(100);

Promises

A Promise ямар нэгэн үйлдэл дуусахад , эсвэл бүтэлгүйтэхийг илэрхийлдэг object юм аа

Үүсгэх


const myPromise = new Promise((resolve, reject) => {
  let success = true;
  if (success) {
    resolve("Success!");
  } else {
    reject("Error occurred.");
  }
});

Using .then() and .catch()

Өмнөхөөсөө ялгаатай нь .then() нь Chaining хийхэд амар болгочихож байна гэж ойлгоод байгаа 

myPromise
  .then(result => {
    console.log("Success:", result);
  })
  .catch(error => {
    console.error("Error:", error);
  });

Chaining Promises

Promise chaining гэдэг нь нэг promise амжилттай биелсний дараа дараагийн promise руу шилжих гэсэн үг.Хэрэглэснээр код илүү засахад амар , эмх цэгцтэй болдог. 

fetch('https://api.example.com/user')
    .then(response => response.json())         // 1-р promise
    .then(user => {
        console.log(user);
        return fetch(`https://api.example.com/posts?user=${user.id}`);
    })                                         // 2-р promise
    .then(response => response.json())
    .then(posts => {
        console.log('Нийтлэлүүд:', posts);       // 3-р promise
    })
    .catch(error => {
    console.error('Алдаа:', error);
    });

Promise.all

Promise-ийн нэг функц бөгөөд олон амлалт (promises)-ыг зэрэг (нэгэн зэрэг) гүйцэтгүүлж, бүгд амжилттай дуусахыг нэг дор хүлээдэг. 

const p1 = fetch('/api/user');
const p2 = fetch('/api/products');

Promise.all([p1, p2])
  .then(responses => Promise.all(responses.map(r => r.json())))
  .then(([user, products]) => {
    console.log("User:", user);
    console.log("Products:", products);
  })
  .catch(error => console.error("Error in one of the requests:", error));
